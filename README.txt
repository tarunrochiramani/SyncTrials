Inorder to use this code, do the following
1) Set your config in  resources/sampleconfig.properties
2) Rename resources/sampleconfig.properties to config.properties


For Group DN's
1) Enter exact Group DN
2) For multiple group search, separate the Group DN's with a ';' eg groupDn=dn1;dn2

For MongoDB on MAC
1) brew install mongodb
2) mongod
